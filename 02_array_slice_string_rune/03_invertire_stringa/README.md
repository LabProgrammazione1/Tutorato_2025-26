# Invertire Stringa

Scrivi un programma che legge da **standard input** una stringa e la stampa invertita (al contrario).

## Problema

Il programma deve:

1. Leggere una stringa da **standard input**.
2. Invertire l'ordine dei caratteri.
3. Stampare la stringa invertita.

## Input/Output

- **Input**: Una riga contenente una stringa (può contenere spazi e caratteri Unicode).
- **Output**: La stringa invertita.

## Esempio

**Input**:

```text
hello
```

**Output**:

```text
olleh
```

## Vincoli

- La stringa può contenere spazi e caratteri Unicode (accentati, emoji, ecc.).
- L'inversione deve operare su caratteri individuali, non su byte.
