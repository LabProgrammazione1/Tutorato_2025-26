# Frequenze

Scrivi un programma che calcoli la frequenza di ogni parola letta stdin.

## Problema

Il programma deve:

1. Leggere parole da stdin fino a EOF (Ctrl+D su Linux/Mac)
2. Calcolare la frequenza di ogni parola

## Input/Output

- **Input**: Una parola per riga, lette fino a fine input (EOF)
- **Output**: Elenco delle frequenze, nel formato (parola: frequenza). L'ordine non è importante

## Esempio

**Input**:

```
ciao
puffo
maiale
ciao
puffo
puffo
puffo
```

**Output**:

```
ciao: 2
puffo: 4
maiale: 1
```
